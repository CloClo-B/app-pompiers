# FEN-Alim 🚒

Bienvenue sur le dépôt GitLab du projet **FEN-Alim** !  
Ce projet est une application développée avec **React Native** et **Expo**.

## Pour démarer :

1. Installer les dépendances

   ```bash
   npm install
   ```

2. Lancer l'application

   ```bash
   npx expo start
   ```


## Pour lancer l'api:

uvicorn main:app --reload --host 0.0.0.0 --port 8000
⚠️⚠️⚠️ bien modifier modifier les fichier le **acceuil** dans (tabs) et le **database** dans le backend avec l'ip de son réseau sinon sa ne fonctionnera pas
