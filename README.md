# FEN-Alim 🚒

Bienvenue sur le dépôt GitLab du projet **FEN-Alim** !  
Ce projet est une application développée avec **React Native** et **Expo**.

## Pour démarer le projet :

1. Installer les dépendances

   ```bash
   npm install
   ```

2. Lancer l'application

   ```bash
   npx expo start
   ```


## Pour lancer l'api et afficher les différents points:

- lancer environement venv en se placant dans : **appli/backend**
- faire : **source api/venv/bin/activate**

ensuite pour lancer l'api faire : 
**uvicorn api.main:app --reload --host 0.0.0.0 --port 8000**
